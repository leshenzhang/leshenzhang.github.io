设置（首次使用先填这里）

怎么拿到令牌？

  1. 打开 github.com/settings → Fine-grained tokens
  2. Repository access 选 Only select repositories，选中你的博客仓库
  3. Permissions → Repository permissions → Contents 设为 Read and write
  4. 生成后复制 github_pat_…，粘贴到上面，点“保存设置”

令牌只保存在你这台设备的浏览器里，不会上传到任何地方。请勿在公用电脑上使用。
